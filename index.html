Safari (iOS · iPhone / iPad)

아이폰 기본 브라우저 · 가장 추천!

⭐
아이폰에서는 Safari가 제일 잘 돼! 크롬 아닌 Safari로 열어줘.
1
Safari로 게임 페이지 열기
아래 버튼 눌러서 게임 주소 복사 후 Safari에 붙여넣기
lve34234-maker.github.io/-rpg--snaily/
2
하단 📤 공유 버튼 탭
Safari 하단 중앙에 있는 네모+위화살표 아이콘 탭
3
홈 화면에 추가 선택
스크롤 내리면 "홈 화면에 추가" 메뉴가 보여. 탭!
4
이름 확인 후 추가
이름은 "공책RPG"로 자동 설정돼. 오른쪽 위 추가 탭!
5
📱 홈 화면에서 아이콘 탭!
이제 전체화면 앱처럼 바로 실행돼 🎉
⚠️
Safari가 아닌 다른 브라우저에서는 "홈 화면에 추가" 옵션이 없을 수 있어!
🔵

Chrome (Android · 안드로이드)

안드로이드 기기에서 설치

✅
안드로이드는 Chrome이 제일 편해! 자동 설치 팝업이 뜨기도 해.
1
Chrome으로 게임 페이지 열기
🔵 lve34234-maker.github.io/-rpg--snaily/
2
우측 상단 ⋮ 메뉴 탭
점 세 개 버튼 탭
3
앱 설치 또는 홈 화면에 추가
버전에 따라 이름이 다를 수 있어. 둘 다 같은 기능!
4
설치 탭
팝업에서 "설치" 버튼 탭. 홈 화면에 아이콘 추가됨!
💡
가끔 주소창 오른쪽에 📲 설치 아이콘이 자동으로 나타나기도 해! 그거 눌러도 돼.
🌐

삼성 인터넷 (갤럭시)

갤럭시 기본 브라우저

1
삼성 인터넷으로 게임 열기
갤럭시 기본 브라우저(파란 지구본 아이콘)로 주소 입력
2
하단 ☰ 메뉴 탭
오른쪽 아래 세 줄 메뉴 버튼 탭
3
➕ 페이지 추가 탭
메뉴에서 "페이지 추가" 선택
4
홈 화면 선택 → 추가
"홈 화면" 선택하면 바탕화면에 아이콘 추가됨!
✅
삼성 인터넷도 전체화면 지원 잘 돼!
💻

PC 설치 (Windows · Mac)

Chrome / Edge 브라우저

💡
PC에서도 앱처럼 설치 가능해! Chrome이나 Edge 추천.
1
Chrome 또는 Edge로 게임 열기
lve34234-maker.github.io/-rpg--snaily/ 📲
2
주소창 오른쪽 📲 설치 아이콘 클릭
아이콘이 안 보이면: 우측 상단 ⋮ 메뉴 → 앱으로 설치
3
설치 클릭
팝업에서 설치 클릭하면 바탕화면에 아이콘 추가!
4
🖥️ 바탕화면 아이콘으로 실행!
별도 창으로 전체화면처럼 실행돼. 작업표시줄에도 핀 가능!
⚠️
Firefox는 PWA 설치를 지원하지 않아. Chrome 또는 Edge를 사용해줘!
